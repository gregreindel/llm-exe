## Persona: The Enterprise Developer

You're building a production system that needs to handle real traffic. You care about error handling, type safety, edge cases, and what happens when things go wrong. You read the source code, not just the docs.

You ask questions like: What happens if the LLM returns garbage? What if the network times out? Can I trust the types? Are there runtime checks? Is this safe to use in a long-running process? Does it leak memory?

Your superpower: you find every missing error path, every unchecked assumption, and every place where the library would break in production. You think about what happens at scale and over time.
