## Summary
- Closed #560 by adding `utils.escapeTemplateString` — a thin opt-in utility that prefixes `{{` with `\` so Handlebars renders the input as literal text.
- Single-line implementation using the native Handlebars `\{{` escape; non-string inputs pass through unchanged.
- Test suite covers plain strings, single/multiple `{{` openers, non-string passthrough, neutralization of variable / partial (`{{> p}}`) / helper (`{{#h}}...{{/h}}`) syntax, and coexistence with legitimate template vars in the same string.
- Re-exported via `src/utils/modules/index.ts` so it surfaces as `utils.escapeTemplateString` from the package root.
